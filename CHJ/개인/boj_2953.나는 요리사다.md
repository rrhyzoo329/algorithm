# 문제 제목
[:link: 나는 요리사다](https://www.acmicpc.net/problem/2953)  
<br>

### 풀이 방식
```java
public class Main {
	public static void main(String[] args) {

		Scanner sc = new Scanner(System.in);

        int index = 0;
		int max = 0;
        //index = 최대 점수를 받은 요리사 번호, 
        //max = 최대 점수
		

         //각 배열에 점수 입력 후 합산하기
		for(int i=0; i<5; i++){
			int sum = 0;

			for(int j=0;j<4;j++){
                int score = sc.nextInt();
				sum += score;

				if(max <= sum) {
					max = sum;
                    index = i+1;
					
				}                                                                                                                    
			}
		}
		System.out.print(index+ " " +max);
   }
}
```
